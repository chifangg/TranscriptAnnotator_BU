# Segmented Directory

This directory contains segmented transcript files generated by `segmentation.ipynb`. Transcripts are broken down into logical chunks or segments and stored here as separate JSON files organized in subdirectories.

## Directory Structure

```
segmented/
├── TranscriptName1/
│   ├── TranscriptName1_0.json
│   ├── TranscriptName1_1.json
│   ├── TranscriptName1_2.json
│   └── ...
├── TranscriptName2/
│   ├── TranscriptName2_0.json
│   ├── TranscriptName2_1.json
│   └── ...
└── README.md
```
